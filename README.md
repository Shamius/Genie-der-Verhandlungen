# Гений переговоров — предыгровой тест

Короткий тест-настройка перед игрой.

## Что внутри

- `index.html` — полностью готовая страница теста
- отдельный `app.js` не нужен: логика встроена прямо в страницу

## Идея

Это не тест на знание переговорной теории. Это предыгровая настройка: на что человеку особенно важно опираться во время партии, чтобы не сорваться в привычную, но не самую выгодную реакцию.

## Результат

На выходе участник получает один главный настрой перед игрой:

- не клюй на бестактность
- не женись на первой позиции
- сначала вскрывай интересы
- держи паузу, не суетись
- играй не одним ходом, а комбинацией

## Публикация на GitHub Pages

1. Открой `Settings` → `Pages`.
2. В `Build and deployment` выбери `Deploy from a branch`.
3. Укажи ветку `main` и папку `/(root)`.
4. Сохрани.

Через несколько минут GitHub Pages даст публичную ссылку на тест.
